# Hello World This is a basic example of an interactive Javascript Notebook page. [toc] ## Basics **Javascript Notebook** pages are text files that use *Markdown* syntax. The file contains sections of text and Markdown directives, and the file name ends with ".md". When a Notebook page is opened, the file contents are rendered into HTML using the [Showdown][] markdown engine. The file can also contain HTML tags.
